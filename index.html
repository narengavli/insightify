Upper Case
The Upper Case function is used to convert a string to uppercase letters.
Example
Input : Hello World
Output : HELLO WORLD
Lower Case
The Lower Case function is used to convert a string to lowercase letters.
Example
Input : Hello World
Output : hello world
Sentence Case
The Sentence Case function is used to convert the first letter of each sentence in a string to uppercase, and the rest to lowercase.
Example
Input : hello world. i am developer.
Output : Hello world. I am developer.
Title Case
The Title Case function is used to convert the first letter of each word in a string to uppercase, and the rest to lowercase.
Example
Input : hello world
Output : Hello World
Alternating Case
The Alternating Case function is used to alternate the case of each letter in a string.
Example
Input : Hello World
Output : hElLo WoRlD
Line Breaker
A line breaker is a function that splits text into lines so that it fits within a specific width, while maintaining readability. This is done by determining where to break lines in a way that minimizes hyphens and enhances overall visual appearance.
Example
Input :
Do not confuse motion and progress. A rocking horse keeps moving but does not make any progress.
Output :
Do not confuse motion and progress.
A rocking horse keeps moving but does not make any progress.
Paragraph Formatter
The Paragraph Formatter function is used to format a block of text into properly spaced and indented paragraphs. This typically involves adding line breaks and indentation to create visually distinct paragraphs and improve the readability of the text.
Example
Input :
once there was a hungry fox who stumbled upon a vineyard. after seeing the round, juicy grapes hanging in a bunch, the fox drooled. But no matter how high he jumped, he couldn’t reach for it. So he told himself that it was probably sour and left. That night, he had to sleep on an empty stomach.
Output :
 Once there was a hungry fox who stumbled upon a vineyard. After seeing the round, juicy grapes hanging in a bunch, the fox drooled. But no matter how high he jumped, he couldn’t reach for it. So he told himself that it was probably sour and left. That night, he had to sleep on an empty stomach.
Extra Space Remover

Example
Input : Hello   World
Output : Hello World
